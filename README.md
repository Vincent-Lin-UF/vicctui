# Vicctui

A minimal Text User Interface (TUI) for **ComputerCraft / CC: Tweaked**

## Installation

```
pastebin get cbqVz2GU install
install
```

## Controls

- **Up/Down** - Navigate menu
- **Enter** - Select
- **Q** - Quit

## Features

- Command shell access
- Program browser
- Uninstaller
